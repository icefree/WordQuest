# WordQuest 产品需求文档 (PRD)

> **版本**: V1.0  
> **创建日期**: 2025-01-15  
> **产品定位**: 趣味背单词游戏网站  
> **核心理念**: 把学习变成娱乐，让背单词像玩游戏一样上瘾

---

## 1. 产品概述

### 1.1 产品愿景

将枯燥的单词记忆转化为沉浸式冒险体验，通过深度游戏化、即时反馈和社交互动，让用户在"玩"中不知不觉掌握词汇。

### 1.2 目标用户

| 用户群体 | 年龄段 | 核心需求 | 使用场景 |
|---------|--------|---------|---------|
| 小学生 | 8-12岁 | 趣味启蒙、培养兴趣 | 家长监督下学习、课后复习 |
| 初高中生 | 13-18岁 | 应试提分、词汇量扩展 | 备考、日常积累 |
| 大学生 | 18-24岁 | 四六级/考研/出国 | 碎片时间、系统备考 |
| 职场人士 | 24-35岁 | 职业发展、自我提升 | 通勤、午休 |

### 1.3 核心价值主张

- **🎮 游戏化**: 地下城闯关、PvP对战、农场养成，多元玩法
- **⚡ 即时反馈**: 每次操作都有视觉/听觉反馈，成就感拉满
- **🧠 科学记忆**: 基于艾宾浩斯曲线的智能复习算法
- **👨‍👩‍👧 家长友好**: 少儿模式、学习报告、时长控制

## 2. 核心游戏模块

### 2.1 单词地下城 (Word Dungeon)

**核心玩法**: 词汇数据库构建成地下城楼层，玩家通过正确拼写/选择释义来攻击怪物。

#### 战斗机制

```
用户看到中文释义 → 输入英文单词 → 判定正误
    ↓ 正确                    ↓ 错误
对怪物造成伤害              自身受到伤害
触发连击加成                连击清零
获得金币/经验               单词加入"复仇清单"
```

#### 连击系统 (Combo)

| 连击数 | 伤害倍率 | 视觉效果 |
|-------|---------|---------|
| 2-4 | 1.5x | 屏幕边缘轻微火焰 |
| 5-9 | 2.0x | 火焰加剧 + 背景变色 |
| 10-19 | 3.0x | 全屏燃烧效果 + 音效加速 |
| 20+ | 5.0x | 超级模式 + 特殊音乐 |

#### Boss 战

- 每 10 层出现 Boss
- Boss = 用户最常错的"顽固单词"
- 击败 Boss 获得稀有奖励

### 2.2 填词游戏 (Word Puzzles)

六种游戏类型，满足不同学习场景：

| 类型 | 玩法 | 适合场景 |
|-----|-----|---------|
| **经典填字** | 网格填字，根据释义填入单词 | 系统复习 |
| **情境填空** | 文学/新闻/歌词中填入缺失单词 | 语境记忆 |
| **对话补全** | 模拟购物/旅行/面试场景对话 | 口语应用 |
| **故事接龙** | AI生成故事，关键词挖空 | 趣味阅读 |
| **限时挑战** | 60秒内填对尽量多的单词 | 冲刺训练 |
| **协作填字** | 多人实时完成大型填字谜 | 社交互动 |

#### 难度分层

| 难度 | 提示内容 | 分数倍率 |
|-----|---------|---------|
| 简单 | 首字母 + 长度 + 中文 | 1.0x |
| 普通 | 长度 + 中文 | 1.5x |
| 困难 | 仅中文释义 | 2.0x |
| 地狱 | 仅英文释义 | 3.0x |

### 2.3 单词农场 (Word Farm)

将学习进度可视化为植物生长：

```
首次学习 → 播下种子 🌱
复习1-2次 → 发芽 🌿
复习3-5次 → 成长 🌳
完全掌握 → 开花收获 🌸 → 进入"词汇花园"
7天未复习 → 枯萎警告 🥀
```

**互动机制**:
- 每日浇水次数有限，需选择优先复习的单词
- 开花的植物可以收获，获得金币奖励
- 农场美观度随词汇量提升

### 2.4 单词对决 (Word PvP)

**匹配机制**: 根据段位 + 词库等级匹配对手

**对战规则**:
1. 双方同时看到释义
2. 先正确拼写者得分
3. 拼错扣自己血量
4. 先归零者失败

**段位系统**:

| 段位 | 分数范围 | 赛季奖励 |
|-----|---------|---------|
| 学徒 | 0-999 | 基础头像框 |
| 新手 | 1000-1499 | 稀有头像框 |
| 熟练 | 1500-1999 | 专属称号 |
| 精英 | 2000-2499 | 限定皮肤 |
| 大师 | 2500-2999 | 传说皮肤 |
| 宗师 | 3000-3499 | 赛季徽章 |
| 传奇 | 3500+ | 全套奖励 |

### 3.1 反馈矩阵

| 事件 | 视觉效果 | 音效 | 数据变化 |
|-----|---------|-----|---------|
| 答对 | 绿色粒子爆炸 + 分数飞出 | 清脆打击音 | +经验 +金币 |

## 5. 智能学习系统

### 5.1 AI复习算法

基于艾宾浩斯曲线 + 个人行为数据：

```
记忆强度 = f(答题速度, 错误次数, 复习间隔, 单词难度)

IF 记忆强度 < 0.3 THEN 优先复习
IF 记忆强度 > 0.8 THEN 延长间隔
```

**功能**:
- 每日生成"今日必复习"清单(20-50词)
- 错误≥5次的单词进入"顽固词库"，增加出现频率
- 智能调整复习间隔：1天 → 3天 → 7天 → 15天 → 30天

### 5.2 情境记忆辅助

- **AI趣味例句**: 结合热点话题/梗生成有趣例句
- **用户记忆技巧**: 用户分享记忆方法，他人可查看点赞
- **词根词缀拆解**: 自动显示单词结构，帮助理解

## 7. 少儿模式 & 家长监护

### 7.1 少儿专属界面

- **可爱角色**: 卡通动物、萌系精灵（降低对抗性）
- **简化UI**: 减少复杂功能，突出核心学习路径
- **语音辅助**: 全程语音引导，自动播放单词发音
- **图文记忆**: 每个单词配精美插图，视觉联想
- **趣味动画**: 答对触发角色跳舞、放烟花等

### 7.2 少儿专属玩法

| 玩法 | 描述 |
|-----|-----|
| **字母拼图** | 拖拽字母块拼出完整单词 |
| **单词配对** | 卡片矩阵中找单词-图片/中文配对 |
| **听音辨词** | 播放发音，选择正确单词 |
| **单词涂色** | 答对解锁涂色页，完成后可保存分享 |
| **宠物养成** | 用学习获得的爱心喂养虚拟宠物 |

### 7.3 家长监护功能

| 功能 | 描述 |
|-----|-----|
| **账号绑定** | 微信/手机号关联孩子账号 |
| **时长限制** | 设置每日学习上限(建议30-60分钟) |
| **学习报告** | 每日/每周自动推送到家长微信 |
| **内容过滤** | 少儿模式关闭社交聊天功能 |
| **休息提醒** | 每20分钟强制休息提醒(护眼) |
| **奖励审批** | 家长设置真实奖励(零花钱/出游)对应学习目标 |

## 10. 技术架构概要

### 10.1 技术栈

| 层级 | 技术选型 |
|-----|---------|
| 前端框架 | Next.js 14 (App Router) + TypeScript |
